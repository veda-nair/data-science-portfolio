DP-900: Microsoft Azure Data Fundamentals

The DP-900 exam, also known as the Microsoft Azure Data Fundamentals exam, is a certification exam that evaluates fundamental knowledge of data concepts and core data services on the Microsoft Azure cloud platform. The exam covers essential data-related topics, including data storage options, data processing technologies, data security, and data visualization.

DP-100: Designing and Implementing a Data Science Solution on Azure

The DP-100 exam, also known as the Designing and Implementing a Data Science Solution on Azure exam, is a certification exam that assesses the skills and knowledge required to design and implement end-to-end data science solutions using Microsoft Azure services. The exam covers topics such as data exploration, data preparation, model development, model training, and model deployment on the Azure cloud platform.

Projects

Formula 1 Predictor

I have been a fan of Formula 1 since I was a kid. As a passionate fan of the sport, I thought it would be interesting to predict the likelihood of a certain driver to win a Grand Prix and compare it to the bookmakers’ odds. This project has been divided into three sections: data collection, data analysis and ML modeling. Data was gathered from two sources: the Ergast F1 data repository and the official Formula 1 website. A basic data analysis about the safety of the tracks, significance of pole position etc was done. For predicting the final results of the race, I used both regression and classification methods.

E-commerce sentiment analysis

This project is about exploring topic modelings and unsupervised machine learning by analyzing e-commerce clothings reviews. I have used Natural Language Processing (NLP) and Unsupervised Machine Learning for topic Modeling and “separation” of good and bad reviews using clustering. The dataset used is a women clothings dataset of a real e-commerce business.

Clustering of news headlines in India

In this project, I have used k-means clustering and vectorization to create clusters of news headlines. The dataset is a persistent historical archive of noteable events in the Indian subcontinent from start-2001 to q1-2022, recorded in real-time by the journalists of India. It contains approximately 3.6 million events published by Times of India. A majority of the data is focusing on Indian local news including national, city level and entertainment.
